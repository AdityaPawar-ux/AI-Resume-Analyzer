# AI Resume Analyzer

## Overview

AI Resume Analyzer is a web-based application developed using Flask and MySQL that helps users analyze their resumes and calculate ATS (Applicant Tracking System) scores.

The system extracts skills from uploaded resumes, calculates ATS scores, provides feedback, recommends suitable job roles, and allows administrators to monitor analytics through dashboards and charts.

---

## Features

* User Registration and Login
* Admin Login and Management
* Resume Upload (PDF/DOCX)
* Skill Extraction
* ATS Score Calculation
* Resume Feedback
* Job Recommendations
* PDF Report Generation
* Resume Analytics Dashboard
* Pie Chart Visualization
* Bar Graph Visualization
* User Management Panel

## Installation

1. Clone Repository

2. Install Dependencies

pip install -r requirements.txt

3. Create MySQL Database

Database Name:

resume_analyzer

4. Run Application

python app.py

5. Open Browser

http://127.0.0.1:5000
